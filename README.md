### 통계적 기본 의미

| 경우                              | 명칭         | 계산식                                 |
| --------------------------------- | ------------ | -------------------------------------- |
| 지각할 확률                       | 사전 확률    | $P[E = 1]$                             |
| 버스를 타고 지각할 확률           | 결합 확률    | $P[E=1  \cap T=BUS ]$                  |
| 버스를 탔을 때 지각할 확률        | 사후 확률    | $\dfrac {P[E=1 \cap T=BUS]}{p[T=BUS]}$ |
| 지각을 했을 때 버스를 탔을 가능성 | 가능도(우도) | $\dfrac {P[T=BUS \cap E=1]}{P[E=1]}$   |
